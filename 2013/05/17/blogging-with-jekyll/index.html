Blogging With Jekyll

May 17th, 2013 | Comments

Finally, I have decided to use Jekyll for my personal blog on Information Security and have chosen Octopress, thinking I would spent less effort for design/ configuration tweaks ;).

To be honest, I didn’t like the default theme (no offense if you are using one), so after googling around I have found Whiterspace theme and tweaked it a bit.

It took me less than a day to setup everything, configuring and rewriting some parts of the code to meet my design tastes. I didn’t have any previous CSS or Rails experience before setting up this blog, so I think it’s a useful time to spend. What do you think?
